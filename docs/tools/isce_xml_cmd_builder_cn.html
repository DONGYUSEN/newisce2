ISCE2 XML 与命令行生成器

三标签页：`GRD`、`干涉处理`、`Stack`。已按 ISCE2 支持的传感器列表提供选择，并在底部加入远程终端辅助窗口。

GRD 处理（rtcApp）

支持 GRD 传感器：`SENTINEL1`、`RADARSAT2`、`TERRASARX`。

格式：`start:end`，可留空

干涉处理（topsApp / stripmapApp）

TOPS 模式自动限制为 `SENTINEL1`；Stripmap 模式可选 ISCE2 Stripmap 支持传感器（含 Tianyi/Lutan/ALOS4/Capella）。

TOPS/Stripmap 均可留空

Stack 处理（stackSentinel / stackStripMap）

TOPS Stack 是 Sentinel-1 专用。Stripmap Stack 支持按 `-S` 选择 Stripmap 传感器。

远程终端窗口（SSH + WebTerm）

浏览器不能直接原生 SSH。此区域提供 SSH 命令生成、Web 终端嵌入（如 ttyd/wetty）和本地模拟命令窗。

例如服务器运行: `ttyd -p 7681 bash -l`
若无法显示，通常是服务器未部署 Web 终端服务，或浏览器/反向代理限制了 iframe / 跨域策略。
